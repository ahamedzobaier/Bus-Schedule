# Bus Schedule

## Introduction

**UU-Bus: Centralized Campus Transit System**

A full-stack solution designed to streamline university bus operations. This application replaces manual scheduling with a dynamic digital platform where students can view real-time routes and administrators can manage the fleet.

### Key Features

- **Dynamic Scheduling** - Real-time table updates with campus direction filtering
- **Admin Management** - CRUD operations for routes, drivers, and student accounts
- **Mobile-First Design** - Responsive UI optimized for on-the-go schedule checks
- **Safety & Notifications** - Integrated emergency hotlines and automated transit alerts
